  2. 사용자가 모델에게 주는 질문이나 지시
  3. 모델 설치 파일
  4. 그래프 제목

  3. AI가 이미지 크기를 키우는 기능
  4. AI가 파일명을 바꾸는 기능
 1 할루시네이션은 사실과 다르지만 그럴듯하게 들리는 내용을 생성하는 문제다. 2
5 쉬움AI 기본 개념 멀티모달 모델의 특징으로 가장 적절한 것은 무엇인가?

  2. 오직 숫자 데이터만 읽는다
  3. 항상 인터넷 없이 최신 뉴스를 안다
  4. 반드시 음성만 출력한다
 1 멀티모달 모델은 텍스트뿐 아니라 이미지 등 여러 형태의 입력을 함께 다룰 수 있다. 2
6 쉬움AI 기본 개념 컨텍스트 윈도(context window)는 보통 무엇을 의미하는가?

  3. 실험실 창문 크기
  4. 파일의 저장 경로
 1 컨텍스트 윈도는 모델이 현재 요청에서 함께 읽고 참조할 수 있는 입력 길이 범위를 뜻한다. 2
7 쉬움AI 기본 개념 토큰(token)에 대한 설명으로 가장 적절한 것은 무엇인가?

  2. 모델이 텍스트를 처리하는 단위다
  3. 실험 횟수의 단위다
  4. 엑셀 시트 이름이다
 2 토큰은 단어, 부분 단어, 기호 등을 포함하는 모델의 처리 단위다. 2
8 쉬움취업 준비와 AI 취업 준비 과정에서 AI 활용 역량이 중요해지는 이유로 가장 적절한 것은 무엇인가?
  1. 앞으로 모든 회사가 사람을 뽑지 않기 때문에
  2. 반복 작업을 줄이고 자료 정리·분석·문서 작성 효율을 높일 수 있기 때문에
  3. AI를 쓰면 전공 지식이 전혀 필요 없기 때문에
  4. AI를 쓰면 무조건 면접에 합격하기 때문에
 2 실무에서는 AI를 통해 반복 업무를 줄이고 생산성을 높이는 역량이 점점 중요해지고 있다. 2
9 쉬움취업 준비와 AI 이공계 대학생이 취업 준비에서 AI를 가장 현실적으로 활용할 수 있는 예는 무엇인가?
  1. 실험 결과 요약 초안 작성
  2. 전공 공부를 완전히 대신하게 하기
  3. 모든 과제를 무검토로 제출하기
  4. 시험 부정행위를 자동화하기
 1 요약, 초안 작성, 자료 정리 같은 보조 업무는 학생이 AI를 현실적으로 활용하기 좋은 예다. 2
10 쉬움취업 준비와 AI 회사가 AI 활용 능력을 평가할 때 단순히 '툴을 써봤다'보다 더 중요하게 보는 것은 무엇인가?
  1. 툴 이름을 많이 외우는 것
  2. 문제를 정의하고 결과를 검증하며 업무에 연결하는 능력
  3. 하루에 프롬프트를 몇 번 썼는지
  4. 모델 이름을 발음하는 능력
 2 실무에서는 도구 사용 경험보다 문제 해결 과정과 검증 능력이 더 중요하다. 2
11 쉬움AI 오해 바로잡기 'AI가 답했으니 무조건 맞다'는 생각이 위험한 이유는 무엇인가?
  1. AI는 언제나 농담만 하기 때문에
  2. AI는 틀릴 수 있고, 근거 없는 내용을 그럴듯하게 말할 수 있기 때문에
  3. AI는 숫자를 전혀 읽지 못하기 때문에
  4. AI는 한국어를 이해하지 못하기 때문에
 2 AI는 유용하지만 오류 가능성이 있으므로 결과 검토가 필요하다. 2
12 쉬움AI 오해 바로잡기 'AI를 잘 쓴다'는 말에 가장 가까운 것은 무엇인가?
  1. AI 답변을 그대로 복사해 제출한다
  2. AI에게 맡길 일과 사람이 검토할 일을 구분한다
  3. 전공 지식 없이도 모든 문제를 푼다
  4. 항상 가장 긴 답변을 받는다
 2 AI를 잘 쓴다는 것은 적절히 맡기고, 결과를 사람이 책임 있게 검토하는 것을 포함한다. 2
13 쉬움AI 오해 바로잡기 'AI를 쓰면 코딩을 하나도 몰라도 된다'는 말에 대한 가장 적절한 반응은 무엇인가?
  1. 맞다. 기본 개념은 전혀 필요 없다
  2. 아니다. 기본 개념을 알수록 AI 결과를 더 잘 검토하고 활용할 수 있다
  3. 코딩은 이제 법적으로 금지된다
  4. AI는 코드를 전혀 만들지 못한다
 2 기본 개념이 있어야 AI가 만든 코드나 분석 결과를 판단하고 수정할 수 있다. 2
14 쉬움실무 태도 AI를 사용해 보고서 초안을 만들었을 때 가장 바람직한 다음 단계는 무엇인가?

  2. 수치, 표현, 결론이 맞는지 다시 확인한다
  3. 문서 제목만 바꾸고 끝낸다
  4. 모든 표를 삭제한다

  4. 결론부터 먼저 쓴다
 1 열 이름, 단위, 형식 정리는 이후 분석과 재현성의 기초가 된다. 2

  1. 가장 자주 나온 값
  2. 가운데 있는 값
  3. 전체 값을 더해 개수로 나눈 값
  4. 가장 큰 값과 가장 작은 값의 차이
 3 평균은 대표값의 하나로, 전체 값을 더해서 데이터 개수로 나눈 값이다. 2

  1. 정렬했을 때 가운데 오는 값
  2. 가장 많이 나온 값
  3. 모든 값의 합
  4. 분산의 제곱근
 1 중앙값은 데이터를 정렬했을 때 가운데 위치하는 값이다. 2
19 쉬움데이터 처리 기초 표준편차가 큰 데이터의 특징으로 가장 적절한 것은 무엇인가?

  2. 값들의 퍼짐 정도가 크다
  3. 항상 평균과 중앙값이 같다
  4. 데이터 개수가 반드시 많다
 2 표준편차는 데이터가 평균 주변에서 얼마나 퍼져 있는지를 보여준다. 2
20 쉬움데이터 처리 기초 산점도(scatter plot)는 언제 가장 유용한가?
  1. 두 변수 사이의 관계를 볼 때
  2. 문장을 번역할 때
  3. 파일 압축률을 계산할 때
  4. 코드를 설치할 때
 1 산점도는 예를 들어 온도와 수율처럼 두 변수의 관계를 볼 때 유용하다. 2
21 중간AI 활용 기본기 AI에게 더 좋은 답을 받기 위해 프롬프트를 개선하는 방법으로 가장 적절한 것은 무엇인가?

  3. 조건 없이 '알아서 해줘'만 쓴다
  4. 같은 문장을 무작정 여러 번 붙여 넣는다

25 중간AI 활용 기본기 AI를 사용할 때 '질문을 잘하는 능력'이 중요한 이유는 무엇인가?
