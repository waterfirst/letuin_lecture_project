4. 단계별 수행 방법4. Step-by-Step Guide

1
데이터 탐색Data Exploration

CSV를 pandas로 로드하고, .info(), .describe(), 결측치 확인으로 데이터 구조를 파악합니다.Load CSV with pandas, check structure with .info(), .describe(), and missing-value analysis.

2
분석 질문 설정Define Analysis Questions
  • chamber별로 thickness_error_nm 분포가 다른가?Does thickness_error_nm distribution differ by chamber?
  • 특정 x/y 위치에 결함이 몰리는가?Do defects cluster at specific x/y positions?
  • defect_type 중 누적 80%를 차지하는 상위 결함은?Which top defect_types account for 80% cumulative?
  • yield_score에 유의한 영향을 주는 공정 조건은?Which process conditions significantly affect yield_score?
3
시각화 5종+ 생성Create 5+ Visualizations
  • boxplot — chamber/zone별 thickness_error_nmthickness_error_nm by chamber/zone
  • scatter plot — 공정 조건 vs yield_scoreProcess conditions vs yield_score
  • heatmap / x/y map — 좌표별 thickness_error_nmthickness_error_nm by coordinate
  • Pareto chart — defect_type별 누적 비율Cumulative ratio by defect_type
  • correlation heatmap — 수치 컬럼 상관관계Numeric column correlations
4
통계 분석Statistical Analysis

statsmodels OLS로 yield_score 회귀분석을 수행합니다. p-value가 0.05 미만인 변수를 "유의한 인자"로 표시합니다.Run OLS regression on yield_score using statsmodels. Mark variables with p-value < 0.05 as "significant factors."

5
인사이트 해석Insight Interpretation

각 그래프와 통계 결과에서 공정 엔지니어 관점의 해석을 작성합니다. 개선 제안을 3개 이상 포함합니다.Write process-engineer perspective interpretations for each chart and statistical result. Include 3+ improvement suggestions.

6
HTML 보고서 작성Write HTML Report

그래프, 해석, 통계 결과, 개선 제안을 하나의 HTML 보고서에 정리합니다.Compile charts, interpretations, statistical results, and improvement suggestions into one HTML report.

7
프롬프트 기록 정리Document Prompts

분석 과정에서 사용한 Gemini 프롬프트를 기록으로 남깁니다.Record all Gemini prompts used during analysis.

Q. p-value가 뭔지 잘 모르겠어요Q. I don't understand p-values

A. Gemini에게 "p-value를 공정 엔지니어가 이해할 수 있게 해석해줘"라고 요청하세요. 단, AI 답변을 그대로 믿지 말고 그래프와 도메인 상식으로 다시 확인하세요.A. Ask Gemini to "explain p-value so a process engineer can understand." But don't blindly trust the AI — verify with charts and domain knowledge.

Q. 그래프를 많이 넣으면 높은 점수를 받나요?Q. Do more charts mean higher scores?
