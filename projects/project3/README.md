# 프로젝트 3 — 입사 1일차 비밀 무기: 나만의 엔지니어 AI 툴킷 (최종)

> **렛유인 AI + 바이브코딩 강의** | 최종 프로젝트 (2일)

---

## 개요

이 강의에서 배운 모든 기술을 **3가지 이상 조합**하여
실제 첨단산업 현장에서 **입사 1일차부터 바로 쓸 수 있는** AI 업무 자동화 도구를 만드는 최종 프로젝트입니다.

P1(데이터 분석기) + P2(AI 보고서)에서 쌓은 경험을 바탕으로,
자유 주제로 **나만의 엔지니어링 솔루션**을 완성합니다.

- **수행 기간**: 최종 프로젝트 주말 2일 (토~일)
- **사용 도구**: Gemini API + Streamlit + Telegram Bot + 자유 조합 (3가지 이상)
- **결과물**: 실무에서 바로 작동하는 수준의 AI 엔지니어링 앱

---

## P1 → P2 → P3 성장 여정

```
P1: 데이터 분석기        → 차트 시각화 (Streamlit + Plotly)
P2: 자동 보고서 앱       → AI 인사이트 + 보고서 자동화 (+ Gemini API)
P3: 나만의 AI 툴킷 (종합) → 모니터링 + 알림 + AI 분석 통합 (+ Telegram + 자유)
```

| 역량 | P1 | P2 | P3 (이번) |
|------|----|----|-----------|
| 시각화 | ✅ 기초 | ✅ 심화 | ✅ 자유 |
| AI API | ❌ | ✅ Gemini | ✅ Gemini + 확장 |
| 보고서 | ❌ | ✅ 자동 생성 | ✅ 자동 + 전송 |
| 알림/자동화 | ❌ | ❌ | ✅ **텔레그램 봇** |
| 배포 | ❌ | 선택 | ✅ **필수** |
| 설계 | 가이드 따라 | 프롬프트 직접 | **전체 직접 설계** |

## 난이도별 수행 가이드

### 🟢 하 (기본) — 공정 모니터링 알림 시스템

> **목표**: 센서 데이터 대시보드 + 임계값 초과 시 텔레그램 알림 전송

**소요 시간**: 3~4시간

**제공되는 것**:
- Gemini 프롬프트 전문
- 텔레그램 봇 생성 가이드
- 단계별 실행 가이드

**Step 1.** 텔레그램 봇 만들기:
```
1. Telegram에서 @BotFather 검색
2. /newbot 입력
3. 봇 이름 입력 (예: MyProcessBot)
4. 봇 username 입력 (예: my_process_monitor_bot)
5. 발급된 BOT_TOKEN 복사
6. 봇에게 아무 메시지 보내기
7. https://api.telegram.org/bot{BOT_TOKEN}/getUpdates 에서 chat_id 확인
```

**Step 2.** Gemini에 아래 프롬프트를 붙여넣으세요:

```
Streamlit으로 공정 모니터링 대시보드를 만들어줘.

기능:
1. 가상 센서 데이터 시뮬레이션 (온도, 압력, 두께)
   - 1초마다 새 데이터 생성 (정상 범위 + 가끔 이상치)
2. 실시간 게이지 차트 3개 (현재 온도, 압력, 두께)
3. 최근 50개 데이터 라인 차트
4. 상태 표시: 정상(🟢), 주의(🟡), 위험(🔴)
   - 온도 > 370: 위험, > 360: 주의
   - 압력 > 2.8: 위험, > 2.7: 주의
5. 위험 상태일 때 텔레그램으로 알림 전송
   - requests 라이브러리로 Telegram Bot API 직접 호출
   - 메시지: "⚠️ [위험] 온도 380°C 감지 — 즉시 점검 필요"
6. 사이드바: BOT_TOKEN, CHAT_ID 입력 필드
7. 모든 텍스트 한국어

telegram 알림은 requests 라이브러리의 POST로 구현해줘:
url = f"https://api.telegram.org/bot{token}/sendMessage"
```

**Step 3.** `streamlit run app.py` → 사이드바에 BOT_TOKEN, CHAT_ID 입력 → 모니터링 시작

**평가 기준**:

- [ ] 상태 판단 로직 동작 (정상/주의/위험)
- [ ] 텔레그램 알림 수신 확인 (스크린샷)
- [ ] 스크린샷 1장 (앱) + 1장 (텔레그램 알림)

## 텔레그램 봇 빠른 시작 가이드

### 1. 봇 생성
```
@BotFather → /newbot → 이름 입력 → username 입력 → TOKEN 받기
```

### 2. Chat ID 확인
```
봇에게 아무 메시지 보내기
→ https://api.telegram.org/bot{TOKEN}/getUpdates
→ result → message → chat → id 값 복사
```

### 3. 메시지 보내기 (Python)
```python
import requests

def send_telegram(token, chat_id, message):
    url = f"https://api.telegram.org/bot{token}/sendMessage"
    requests.post(url, json={
        "chat_id": chat_id,
        "text": message,
        "parse_mode": "Markdown"
    })
```
