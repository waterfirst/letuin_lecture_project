# 프로젝트 1 — Gemini로 만드는 나만의 공정 데이터 분석기

> **렛유인 AI + 바이브코딩 강의** | CH.1 수료 후 첫 번째 프로젝트 (2일)

---

## 개요

Gemini AI에게 대화하듯 요청하면서 **제조 공정 데이터를 자동 분석하는 웹 앱**을 만드는 프로젝트입니다.
코딩 경험이 전혀 없어도 AI 프롬프트만으로 실무형 데이터 분석 도구를 완성할 수 있습니다.

- **수행 기간**: CH.1 이후 주말 2일 (토~일)
- **사용 도구**: Gemini AI + Python Streamlit + Plotly
- **결과물**: 가상 공정 데이터를 업로드하면 자동으로 통계/시각화가 나오는 웹 앱

---

## 학습 목표

1. Gemini AI에게 데이터 분석 코드를 요청하는 프롬프트 기술을 익힌다
2. Streamlit으로 누구나 쓸 수 있는 웹 앱을 만든다
3. 제조 공정 데이터(온도, 압력, 두께, 수율 등)를 시각화하고 해석한다
4. AI Tool 활용(70%)과 도메인 지식(30%)의 균형을 체험한다

## 관심 산업 선택 (하나 이상)

수강생은 자신의 관심 분야에 맞는 가상 데이터로 프로젝트를 수행합니다.

| 산업 | 데이터 예시 | 핵심 분석 |
|------|-----------|----------|
| **반도체** | 웨이퍼 CD, 온도, 압력, 식각률 | 균일도 분석, 이상 탐지 |
| **디스플레이** | 불량 좌표, 설비 파라미터, 수율 | 히트맵, Pareto 분석 |
| **2차전지** | 충방전 전압, 용량, 온도 사이클 | 열화 추이, 수명 예측 |
| **범용 제조** | 온도, 압력, 두께, 수율 | SPC 관리도, 상관관계 |

> 제공되는 합성 데이터는 "범용 제조" 기준이며, 다른 산업으로 변경 가능합니다.

## 난이도별 수행 가이드

### 🟢 하 (기본) — Gemini 프롬프트 복붙 → 바로 실행

> **목표**: AI가 만들어 준 코드를 실행해서 결과를 확인하는 것이 핵심. 과정을 경험합니다.

**소요 시간**: 3~4시간

**제공되는 것**:
- 합성 데이터 CSV 파일 (`sample_data.csv`)
- Gemini 프롬프트 전문 (복사-붙여넣기용)
- 실행 명령어 가이드

**단계별 가이드**:

**Step 1.** [Gemini](https://gemini.google.com)에 접속합니다.

**Step 2.** 아래 프롬프트를 복사해서 붙여넣으세요:

```
나는 제조 공정 데이터를 분석하는 Streamlit 웹 앱을 만들려고 해.

아래 조건으로 app.py 파일을 만들어줘:

1. CSV 파일 업로드 기능 (st.file_uploader)
2. 업로드된 데이터의 기초 통계 테이블 표시 (평균, 표준편차, 최소, 최대)
3. Plotly로 각 수치 컬럼의 히스토그램 그리기
4. Plotly로 시간순 라인 차트 그리기
5. 사이드바에 컬럼 선택 필터 추가
6. 모든 텍스트는 한국어로

CSV 컬럼 예시: timestamp, temperature, pressure, thickness, yield_rate

requirements.txt도 함께 만들어줘.
```

**Step 3.** Gemini가 생성한 `app.py` 코드를 복사해서 파일로 저장합니다.

**Step 4.** 터미널에서 실행합니다:
```bash
pip install streamlit pandas plotly
streamlit run app.py
```

**Step 5.** 브라우저에서 `sample_data.csv`를 업로드하고 결과를 확인합니다.

**Step 6.** 실행된 화면을 스크린샷으로 캡처합니다.

**💡 코드가 에러나면?**
```
이 코드를 실행하니 아래 에러가 나와. 고쳐줘:
[에러 메시지를 그대로 복사-붙여넣기]
```

**평가 기준**:
- [ ] 앱이 정상 실행되는가
- [ ] CSV 업로드 후 통계 테이블이 표시되는가
- [ ] 히스토그램 1개 이상 표시되는가
- [ ] 실행 스크린샷 제출

### 🟡 중 (응용) — 프롬프트 직접 작성 + 인터랙티브 차트

> **목표**: 프롬프트를 직접 설계하고, 분석 기능을 확장합니다.

**소요 시간**: 6~8시간

**제공되는 것**:
- 합성 데이터 CSV 파일 (`sample_data.csv`)
- 방향 힌트 (구체적인 프롬프트는 없음)

**요구사항**:
- Plotly 인터랙티브 차트 **3종 이상** (히스토그램, 박스플롯, 산점도, 라인차트 중 선택)
- 사이드바 필터 (컬럼 선택, 날짜 범위, 임계값 슬라이더)
- 이상치 표시 기능 (평균 ± 2σ 벗어난 데이터 하이라이트)
- 프롬프트를 **직접 작성** (템플릿 없음)

**힌트**:
- "Plotly의 `add_hline`을 쓰면 관리 한계선을 그릴 수 있어요"
- "박스플롯은 이상치를 자동으로 표시해줍니다"
- "산점도에서 두 변수의 상관관계를 한눈에 볼 수 있어요"
- Gemini에게 여러 번 수정 요청해도 됩니다: "이 차트에 평균선도 추가해줘", "색상을 공정별로 다르게 해줘"

**프롬프트 작성 팁**:
```
❌ "데이터 분석 앱 만들어줘" (너무 막연)
✅ "50개 센서에서 1000개 샘플을 수집한 공정 데이터를 분석하는 Streamlit 앱을 만들어줘.
    박스플롯으로 각 센서의 분포를 보여주고,
    평균 ± 2σ를 벗어나는 이상치를 빨간색으로 표시해줘." (구체적)
```

**평가 기준**:
- [ ] Plotly 인터랙티브 차트 3종 이상
- [ ] 사이드바 필터 동작
- [ ] 이상치 표시 기능
- [ ] 사용한 Gemini 프롬프트 기록 제출
- [ ] 스크린샷 2장 이상

## 합성 데이터 생성

프로젝트에 사용할 가상 공정 데이터를 Gemini로 직접 생성할 수도 있습니다.

**Gemini 프롬프트 예시**:
```
제조 공정 합성 데이터를 생성하는 Python 코드를 만들어줘.

조건:
- 1000개 샘플, 30일치 시계열 데이터
- 컬럼: timestamp, line (A/B/C), temperature, pressure, thickness, yield_rate
- temperature: 평균 350, 표준편차 5 (정상), 가끔 380 이상 스파이크
- pressure: 평균 2.5, 표준편차 0.1
- thickness: 평균 100nm, 표준편차 2
- yield_rate: temperature, pressure와 상관관계 있게 생성 (온도 높으면 수율 낮음)
- CSV 파일로 저장

pandas와 numpy만 사용해줘.
```

## 자주 묻는 질문

**Q. 코딩을 전혀 모르는데 할 수 있나요?**
A. 네! '하' 레벨은 프롬프트 복붙과 파일 저장, `streamlit run` 명령어 실행만으로 완성됩니다.

**Q. Gemini가 만든 코드가 실행이 안 돼요.**
A. 에러 메시지를 통째로 복사해서 Gemini에게 "이 에러 고쳐줘"라고 보내세요. AI가 수정 코드를 줍니다.

**Q. Streamlit이 뭔가요?**
A. Python으로 웹 앱을 만드는 프레임워크입니다. 복잡한 HTML/CSS 없이 `st.write()`, `st.plotly_chart()` 같은 명령어로 바로 웹 화면을 만들 수 있습니다.

**Q. 데이터를 실제 공정 데이터로 바꿔도 되나요?**
A. '상' 레벨에서는 권장합니다! 단, 회사 기밀 데이터는 사용하지 마세요. 공개 데이터셋을 활용하세요.

**Q. Claude 대신 Gemini를 써야 하나요?**
A. 이 과정에서는 Gemini를 메인 도구로 사용합니다. Gemini가 코드 생성에 매우 강력합니다.
