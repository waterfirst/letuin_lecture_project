초보자가 주말 2일 동안 완성할 수 있도록 재구성한 프로젝트 안내서입니다. 상·중·하 모두 같은 문제를 풀지만, 제공되는 힌트와 프롬프트의 구체성만 다르게 설계했습니다.

1. 프로젝트 한 줄 정의

제조 공정 CSV 데이터를 업로드하면, AI가 만든 Streamlit 앱이 자동으로 기초 통계, 시각화, 이상치, 공정 해석을 보여주는 분석 도구를 만든다.

기존 repo 기준 프로젝트 1은 projects/project1에 있으며, 예제는 examples/하, examples/중, examples/상으로 나뉘어 있습니다. 이 보완안은 기존 구조를 유지하면서 과제 설명, 2일 진행표, 난이도별 힌트 차이를 더 명확히 만든 버전입니다.

2. 공통 문제

상·중·하 모든 수강생은 아래 동일한 문제를 해결합니다.

문제: 공정 데이터 CSV를 업로드하면, 주요 공정 변수의 분포와 시간 변화, 이상치, 수율과 변수의 관계를 한 화면에서 확인할 수 있는 웹 앱을 만든다.

공통 데이터 컬럼

컬럼 의미 분석 관점
timestamp 측정 시간 시간에 따른 공정 안정성 확인
line 생산 라인 A/B/C 라인별 편차 비교
temperature 공정 온도 온도 스파이크와 수율 저하 관계
pressure 공정 압력 압력 안정성, 이상치 탐지
thickness 막 두께 또는 제품 두께 목표값 대비 편차, SPC 관리도
yield_rate 수율 주요 변수와의 상관관계

Gemini 복붙 프롬프트

나는 제조 공정 데이터를 분석하는 Streamlit 웹 앱을 만들려고 해.
코딩 초보자도 실행할 수 있도록 app.py 전체 코드를 만들어줘.

데이터 컬럼은 다음과 같아:
timestamp, line, temperature, pressure, thickness, yield_rate

앱 요구사항:
1. CSV 파일 업로드 기능을 넣어줘.
2. 업로드된 데이터 미리보기 10행을 보여줘.
3. 수치형 컬럼의 평균, 표준편차, 최소값, 최대값을 표로 보여줘.
4. 사용자가 사이드바에서 수치형 컬럼 하나를 고르면 Plotly 히스토그램을 그려줘.
5. timestamp가 있으면 선택한 컬럼의 시간순 라인 차트를 그려줘.
6. yield_rate 평균을 metric 카드로 보여줘.
7. 모든 화면 문구는 한국어로 써줘.
8. 코드 안에 초보자가 이해할 수 있는 주석을 달아줘.

requirements.txt 내용도 함께 알려줘.

실행 명령어

pip install streamlit pandas plotly numpy
streamlit run app.py

6. 중 난이도: 기능 힌트만 제공

목표는 기본 앱을 만든 뒤, Gemini에게 여러 번 수정 요청하면서 분석 기능을 넓히는 것입니다.

필수 결과물

  • Plotly 차트 3종 이상
  • 사이드바 필터 2개 이상
  • 평균 ± 2σ 기준 이상치 표시
  • 사용한 Gemini 프롬프트 기록
  • 스크린샷 2장

기능 힌트

기능 Gemini에게 요청할 방향 구현 키워드
라인별 필터 line 컬럼이 있으면 A/B/C를 선택해서 데이터가 필터링되게 해줘. st.sidebar.multiselect
날짜 범위 필터 timestamp를 날짜로 변환하고 기간을 선택하게 해줘. pd.to_datetime, st.date_input
이상치 표시 선택 컬럼에서 평균 ± 2σ를 벗어난 점을 빨간색으로 표시해줘. mean, std, px.scatter
상관관계 수율과 temperature, pressure, thickness의 상관관계를 보여줘. corr, px.imshow
공정 해석 분석 결과를 엔지니어 관점으로 5줄 요약하게 해줘. st.info, st.warning

중 난이도용 시작 프롬프트

제조 공정 데이터 분석 Streamlit 앱을 만들고 싶어.
아래 컬럼을 가진 CSV를 업로드해서 분석할 거야:
timestamp, line, temperature, pressure, thickness, yield_rate

내가 원하는 기능은 다음과 같아:
- CSV 업로드
- line별 필터
- 날짜 범위 필터
- 히스토그램, 박스플롯, 산점도, 시간순 라인 차트 중 3개 이상
- 선택한 컬럼의 평균 ± 2σ 이상치를 빨간색으로 표시
- 수율과 공정 변수의 상관관계 히트맵

초보자가 수정하기 쉽게 함수 단위로 app.py를 작성해줘.

평가 기준

시각화서로 다른 Plotly 차트 3종 이상이 있다.
상호작용필터를 바꾸면 차트와 통계가 함께 바뀐다.
이상치2σ 기준 이상치가 표 또는 차트에 표시된다.
프롬프트Gemini에게 어떤 요청을 했는지 기록한다.
해석공정 변수와 수율 관계를 5줄 이상 설명한다.

9. 수강생에게 강조할 규칙

회사 실제 공정 데이터, 고객사 데이터, 내부 설비명, API Key, Telegram Token, GitHub Token은 절대 업로드하거나 GitHub에 올리지 않습니다.
  • Gemini가 준 코드는 반드시 한 번 실행해보고 제출합니다.
  • 에러가 나면 에러 메시지를 숨기지 말고 그대로 Gemini에게 전달합니다.
  • 차트가 많다고 좋은 과제가 아닙니다. “그래서 공정상 무엇을 알 수 있는가?”를 써야 합니다.
  • 상·중·하의 차이는 문제 난이도가 아니라 힌트의 양입니다.
