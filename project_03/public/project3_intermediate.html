import requests

def send_telegram(token, chat_id, message):
    url = f"https://api.telegram.org/bot{token}/sendMessage"
    requests.post(url, json={
        "chat_id": chat_id,
        "text": message,
        "parse_mode": "Markdown"
    })

2프로젝트 구조 설계Design Project Structure

힌트: 멀티 페이지 또는 멀티 탭으로 구성하세요. 최소 3개 화면: 데이터 업로드, AI 분석, 보고서. Hint: Use multi-page or multi-tab layout. At least 3 screens: data upload, AI analysis, report.

3데이터 업로드 + 시각화 구현Implement Data Upload + Visualization

힌트: CSV/Excel 업로드 → Pandas로 읽기 → 4종 이상 Plotly 차트 (산점도, 히스토그램, 박스플롯, 히트맵 등) Hint: CSV/Excel upload → Read with Pandas → 4+ Plotly charts (scatter, histogram, boxplot, heatmap, etc.)

4Gemini API 연동 — AI 인사이트Integrate Gemini API — AI Insights

힌트: 데이터 통계 요약을 Gemini에게 전달하고, "이 데이터의 핵심 패턴 3가지와 개선 포인트를 한국어로 설명해줘"라고 요청하세요. Hint: Send data statistics summary to Gemini and ask "Explain 3 key patterns and improvement points in this data in Korean."

5Markdown 보고서 생성 + 다운로드Generate Markdown Report + Download

힌트: AI 인사이트 + 시각화 결과를 Markdown 보고서로 조합하고, st.download_button으로 다운로드 기능을 추가하세요. Hint: Combine AI insights + visualizations into a Markdown report and add download via st.download_button.

6텔레그램으로 요약 전송Send Summary via Telegram

힌트: 보고서 전체가 아닌 핵심 3줄 요약만 텔레그램으로 전송하세요. Gemini에게 "이 분석을 3줄로 요약해줘"라고 요청하면 효과적입니다. parse_mode="Markdown"을 사용하면 메시지가 깔끔하게 표시됩니다. Hint: Send only a 3-line summary (not the full report) to Telegram. Ask Gemini "Summarize this analysis in 3 lines." Using parse_mode="Markdown" makes messages clean.

13. 트러블슈팅 / FAQ13. Troubleshooting / FAQ

트러블슈팅Troubleshooting

텔레그램 봇에서 메시지가 안 옴No message from Telegram bot
봇에게 먼저 /start 메시지를 보내세요. getUpdates에서 chat_id를 재확인하세요.Send /start to the bot first. Re-check chat_id from getUpdates.
ConnectionError 발생ConnectionError
인터넷 연결 확인. 회사/학교 네트워크에서 Telegram API 차단 가능 (VPN 사용).Check internet. Telegram API may be blocked on corporate networks (use VPN).
Gemini API 호출 횟수 초과Gemini API rate limit exceeded
무료 티어 분당 15회 제한. time.sleep(4)로 호출 간격 조절. 데이터 전체가 아닌 요약만 API에 전송.Free tier: 15 calls/min. Use time.sleep(4). Send summary, not full data.
Streamlit Cloud에서 텔레그램이 안 됨Telegram not working on Streamlit Cloud
Streamlit Cloud의 Secrets에 BOT_TOKEN, CHAT_ID를 등록하고 st.secrets["BOT_TOKEN"]으로 접근하세요.Register BOT_TOKEN, CHAT_ID in Streamlit Cloud Secrets. Access via st.secrets["BOT_TOKEN"].
