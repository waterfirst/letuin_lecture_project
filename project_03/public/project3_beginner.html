3Python에서 메시지 보내기Send Message from Python

import requests

def send_telegram(token, chat_id, message):
    url = f"https://api.telegram.org/bot{token}/sendMessage"
    requests.post(url, json={
        "chat_id": chat_id,
        "text": message,
        "parse_mode": "Markdown"
    })

⚠ 봇에게 먼저 메시지를 보내야 봇이 chat_id를 인식합니다.⚠ You must message the bot first so it can recognize your chat_id.

텔레그램 봇에서 메시지가 안 옴No message from Telegram bot
@BotFather에서 봇을 만든 후, 봇에게 먼저 /start 또는 아무 메시지를 보내야 합니다. getUpdates에서 chat_id를 재확인하세요. TOKEN에 공백이나 줄바꿈이 포함되지 않았는지 확인하세요.After creating the bot with @BotFather, you must send /start or any message to the bot first. Re-check chat_id from getUpdates. Ensure TOKEN has no spaces or line breaks.
requests.post 호출 시 ConnectionErrorConnectionError on requests.post
인터넷 연결을 확인하세요. 회사/학교 네트워크에서 Telegram API가 차단되었을 수 있습니다 (VPN 사용).Check your internet connection. Telegram API may be blocked on corporate/school networks (try VPN).
Gemini API 호출 횟수 초과Gemini API rate limit exceeded
무료 티어는 분당 15회 제한입니다. time.sleep(4) 등으로 호출 간격을 조절하세요.Free tier is limited to 15 calls/min. Use time.sleep(4) to throttle calls.
