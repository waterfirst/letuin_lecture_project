사용 방법
===============================================================================

1. Google AI Studio 방법 (무료, 추천):
   - https://aistudio.google.com 접속
   - "Generate image" 또는 Imagen 탭 선택
   - 위 프롬프트 복사 → 붙여넣기
   - Generate 클릭 → 다운로드
   - public/ 폴더에 저장

2. ChatGPT Plus 방법 ($20/월 구독 필요):
   - ChatGPT 접속 (GPT-4 선택)
   - 위 프롬프트 복사 → 붙여넣기
   - 자동으로 DALL-E 3 실행
   - 다운로드 → public/ 폴더에 저장

3. Python 자동화 방법:
   - .env 파일에 OPENAI_API_KEY 추가
   - pip install openai requests python-dotenv
   - python generate_images.py 실행
