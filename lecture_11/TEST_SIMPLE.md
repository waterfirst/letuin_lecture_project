# 간단한 테스트

브라우저에서 **http://localhost:5173/lecture01/** 를 열어보세요.

## 현재 상태

파일 구조를 수정했습니다:
- ✅ src/ 폴더 생성
- ✅ App.tsx, index.css, main.tsx를 src/로 이동

## 다음 단계

1. 브라우저를 **새로고침** (F5 또는 Ctrl+R)
2. 여전히 흰 화면이면 **개발자 도구** 열기 (F12)
3. **Console** 탭에서 에러 메시지 확인

## 예상 에러

만약 에러가 있다면:
- 빨간색 에러 메시지 복사
- 또는 스크린샷

## 간단한 테스트용 App 만들기

만약 계속 문제가 있다면, 간단한 테스트 컴포넌트로 교체하겠습니다.
