9. Troubleshooting / FAQ9. Troubleshooting / FAQ

Q: CSV 파일이 fetch로 로드되지 않아요Q: CSV file won't load with fetch

A: 로컬 파일을 직접 fetch하면 CORS 에러가 발생합니다. python -m http.server 8000으로 로컬 서버를 띄우거나, HTML 파일 내에 CSV 데이터를 직접 임베딩하세요.A: Fetching local files directly causes CORS errors. Run a local server with python -m http.server 8000, or embed CSV data directly in the HTML file.

Q: D3.js Heatmap에서 포인트가 웨이퍼 밖에 찍혀요Q: D3.js Heatmap points appear outside the wafer

A: SVG viewBox와 scale 함수의 domain/range를 확인하세요. 데이터 좌표는 -145~+145mm이고, SVG 좌표계로 올바르게 매핑되어야 합니다. clipPath로 원형 마스크를 적용하세요.A: Check SVG viewBox and scale function domain/range. Data coordinates span -145 to +145mm and must be correctly mapped to SVG coordinates. Apply a circular mask with clipPath.

Q: Chart.js 듀얼 Y축이 안 돼요Q: Chart.js dual Y-axis is not working

A: Chart.js에서 두 번째 Y축은 scales: { y1: { position: 'right' } }로 설정하고, 데이터셋의 yAxisID: 'y1'을 지정해야 합니다.A: Set the second Y-axis with scales: { y1: { position: 'right' } } and assign yAxisID: 'y1' to the dataset in Chart.js.

Q: 보간 후에도 NaN이 남아 있어요Q: NaN values remain even after interpolation

A: 같은 Lot의 모든 값이 이상치인 경우 평균 계산이 불가능합니다. 이 경우 전체 데이터의 정상값 평균으로 대체하거나, 해당 Lot를 분석에서 제외하세요.A: If all values in the same Lot are outliers, mean calculation is impossible. In this case, replace with the overall valid-data mean or exclude that Lot from analysis.
