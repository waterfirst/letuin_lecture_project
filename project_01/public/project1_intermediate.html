프로젝트 1 — 반도체 CVD 박막 두께 Wafer Map 데이터 분석

1. 프로젝트 개요

실제 반도체 양산 공정에서 널리 쓰이는 PECVD 실리콘 산화막(SiOx) 박막 증착 두께 데이터를 분석하는 실무 프로젝트입니다.

엘립소메터(Ellipsometer) 계측 과정에서 발생한 극단적 노이즈(이상치)와 유실된 계측 데이터(결측치)를 정제하고, 웨이퍼 상의 박막 균일도(Uniformity)를 계산하여 비정상 CVD 설비와 공정 이상 원인을 진단합니다.

수행 기간: CH.1 이후 주말 2일 (토~일)
핵심 목표: 데이터 정제(Outlier/NaN 필터링)를 통한 신뢰성 있는 웨이퍼 2D 맵 및 설비 트렌드 시각화와 결함 판정

2. 중급 학습 목표

  1. JavaScript 기반 데이터 파싱 — CSV 파일을 JS로 로드/파싱하여 브라우저 내에서 동적으로 처리하는 능력을 기릅니다.
  2. 이상치 마스킹 알고리즘 구현 — 비정상 이상치를 코드로 마스킹 처리하고, NaN 값은 해당 웨이퍼 정상 계측지 평균으로 보간하는 함수를 작성합니다.
  3. D3.js / Chart.js 동적 시각화 — SVG 또는 Canvas 기반으로 300mm 웨이퍼 규격의 2D Heatmap과 트렌드 차트를 구현합니다.
  4. Antigravity 에이전트 협업 — 로컬 AI 코딩 에이전트와 대화하며 코드를 점진적으로 완성하는 워크플로우를 체험합니다.

3. 사용 데이터셋

모든 수강생은 아래 엘립소메터 49점 계측 데이터를 사용합니다.

CSV 다운로드 링크

  • siox_thickness_data_1.csv — CVD_M02 설비 오목형(Edge-High) 이상 데이터
  • siox_thickness_data_2.csv — CVD_M03 설비 볼록형(Center-High) 이상 데이터

데이터 컬럼 설명

4. 단계별 수행 방법

로컬 에디터에서 Antigravity 에이전트에게 지시하여 결측치 자동 필터링, 아웃라이어 마스킹 알고리즘을 작성하고 인터랙티브 대시보드를 빌드합니다.

1
프로젝트 폴더 및 파일 구조 생성

작업 폴더를 만들고, CSV 파일을 data/ 하위에 넣고, 빈 report_intermediate.html 파일을 생성합니다. Antigravity에게 프로젝트 구조를 잡아달라고 요청합니다.

2
CSV 데이터 로드 및 파싱

Antigravity에게 fetch() + Papa Parse 또는 순수 JS로 CSV를 파싱하는 코드를 요청합니다. 파싱 결과를 JavaScript 배열 객체로 변환합니다.

3
이상치 마스킹 및 결측치 보간 함수 구현

비정상 이상치(-999.0, 999.9, 0.0, 250.3)를 마스킹 처리하고, NaN 값은 해당 웨이퍼의 정상 계측치 평균값으로 채워 넣는 보간 함수를 구현합니다.

4
2D Wafer Map Heatmap 구현

SVG 또는 D3.js 기반으로 300mm 웨이퍼 규격의 2D Wafer Map Heatmap을 그립니다. 두께에 따라 Cyan에서 Red 계열로 HSL 색채가 변화하도록 처리합니다.

  • 웨이퍼 원형 외곽선 (반경 150mm) 표시
  • 49개 계측점을 원형 마커로 표시
  • 두께 값에 따른 연속 색상 매핑 (컬러바 포함)
  • 마우스 호버 시 좌표 및 두께 값 툴팁 표시
5
Lot별 트렌드 타임라인 차트 구현

Lot별 두께 변화와 설비 작동 내역을 함께 보여주는 트렌드 타임라인 차트를 Chart.js 또는 D3.js로 구현합니다.

  • X축: Lot 또는 시간 / Y축: 평균 두께 및 Uniformity
  • CVD 설비별 색상 구분 (M01=파랑, M02=빨강, M03=초록)
  • 이상 구간 하이라이트 영역 표시
6
Uniformity 계산 및 통계 요약 패널

Uniformity (%) = (Max - Min) / (2 * Mean) * 100 공식을 적용하여 설비별/Lot별 계산 결과를 대시보드 상단에 카드형 요약 패널로 표시합니다.

7
최종 통합 및 디자인 정리

모든 차트와 통계를 하나의 report_intermediate.html에 통합합니다. 다크모드 스타일링, 반응형 레이아웃, 인터랙티브 필터(Lot 선택, 설비 선택) 등을 추가합니다.

6. 프롬프트 예시 (Antigravity용)

아래 프롬프트들을 Antigravity 에이전트에게 순서대로 전달하세요.

Step 2: CSV 로드 및 파싱

이 프로젝트 폴더에 data/siox_thickness_data_1.csv 파일이 있어.
이 CSV를 Papa Parse 라이브러리로 파싱해서 JavaScript 배열 객체로 만들어줘.

요구사항:
- HTML 파일 하나에 모든 코드 포함 (CDN으로 라이브러리 로드)
- 파싱 완료 후 콘솔에 전체 행 수, 컬럼 목록, 첫 5행 출력
- thickness_value 컬럼의 기초 통계(평균, 최소, 최대, NaN 개수) 계산해서 화면에 표시

Step 3: 이상치 마스킹 + 결측치 보간

thickness_value에 이상치와 결측치가 있어. 아래 규칙으로 정제 함수를 만들어줘:

1. 이상치 마스킹: -999.0, 999.9, 0.0, 250.3 값은 NaN으로 변환
2. 결측치 보간: NaN 값은 같은 lot_id + cvd_machine 그룹 내의
   정상 thickness_value 평균값으로 대체
3. 정제 전후 통계를 비교하는 요약 카드를 화면 상단에 표시
   (원본 행 수, 이상치 건수, 결측치 건수, 정제 후 평균/표준편차)

Step 4: 2D Wafer Map Heatmap

D3.js로 300mm 웨이퍼 2D Heatmap을 그려줘.

요구사항:
- 웨이퍼 외곽: 반경 150mm 원형 테두리 (회색)
- 계측점: position_x, position_y 좌표에 원형 마커
- 색상: thickness_value에 따라 연속 색상 매핑
  (낮은 두께 = Cyan #00ffff, 높은 두께 = Red #ff0000)
- 컬러바(범례) 하단에 표시
- 마우스 호버 시 툴팁: "X: 30.0mm, Y: -60.0mm, 두께: 1002.3nm"
- Lot 선택 드롭다운: 선택한 Lot의 웨이퍼만 표시
- 다크모드 배경 (#0a0a14)

Step 5: 트렌드 타임라인 차트

Chart.js로 Lot별 트렌드 타임라인 차트를 추가해줘.

요구사항:
- X축: lot_id (LOT-001 ~ LOT-040)
- Y축 (왼쪽): 평균 thickness_value (nm)
- Y축 (오른쪽): Uniformity (%)
- CVD 설비별로 색상 구분 (M01=파랑, M02=빨강, M03=초록)
- 이상치가 많이 발생한 Lot 구간에 배경색 하이라이트
- 다크모드 스타일 (배경 투명, 축/레이블 밝은 색상)

9. 트러블슈팅 (FAQ)

Q. CSV 파일을 fetch()로 로드하면 CORS 에러가 나요

A. 로컬 파일을 직접 fetch하면 CORS 문제가 생깁니다. 해결법:

  • 방법 1: VS Code의 Live Server 확장을 사용하여 로컬 서버로 실행
  • 방법 2: CSV 데이터를 HTML 내 <script> 태그에 직접 임베드
  • 방법 3: Python python -m http.server 8080으로 간이 서버 실행

Q. D3.js Heatmap에서 색상이 제대로 안 나와요

A. 이상치가 포함된 상태에서 색상 스케일 도메인을 설정하면 극단값 때문에 전체 색상이 뭉게집니다. 반드시 이상치 마스킹 후의 정상 데이터로 min/max 도메인을 설정하세요.

// 잘못된 예: 이상치 포함
const colorScale = d3.scaleLinear()
  .domain([d3.min(data, d => d.thickness), d3.max(data, d => d.thickness)])

// 올바른 예: 정제 데이터 사용
const cleanData = data.filter(d => !isNaN(d.thickness) && d.thickness > 100 && d.thickness < 1500);
const colorScale = d3.scaleLinear()
  .domain([d3.min(cleanData, d => d.thickness), d3.max(cleanData, d => d.thickness)])
  .range(["cyan", "red"]);

Q. Chart.js에서 다크모드 배경이 적용이 안 돼요

A. Chart.js의 기본 배경은 투명입니다. Canvas의 부모 div 배경색을 설정하고, 축/레이블 색상을 밝은 색으로 변경하세요.

options: {
  scales: {
    x: { ticks: { color: '#e8e8f0' }, grid: { color: '#1e1e30' } },
    y: { ticks: { color: '#e8e8f0' }, grid: { color: '#1e1e30' } }
  },
  plugins: {
    legend: { labels: { color: '#e8e8f0' } }
  }
}

Q. Uniformity 공식이 음수가 나와요

A. Uniformity 계산은 반드시 이상치를 제거한 후 수행해야 합니다. 이상치(-999 등)가 포함되면 Min이 음수가 되어 비정상적인 값이 계산됩니다.

Q. 웨이퍼 맵에서 좌표가 원 밖으로 벗어나요

A. SVG viewBox와 좌표 스케일을 맞추세요. 웨이퍼 반경은 150mm이므로, 좌표 범위 -150~+150을 SVG 크기에 매핑해야 합니다.
