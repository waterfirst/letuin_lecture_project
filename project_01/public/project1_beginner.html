프로젝트 1 — 반도체 CVD 박막 두께 Wafer Map 데이터 분석

1. 프로젝트 개요

실제 반도체 양산 공정에서 널리 쓰이는 PECVD 실리콘 산화막(SiOx) 박막 증착 두께 데이터를 분석하는 실무 프로젝트입니다.

엘립소메터(Ellipsometer) 계측 과정에서 발생한 극단적 노이즈(이상치)와 유실된 계측 데이터(결측치)를 정제하고, 웨이퍼 상의 박막 균일도(Uniformity)를 계산하여 비정상 CVD 설비와 공정 이상 원인을 진단합니다.

수행 기간: CH.1 이후 주말 2일 (토~일)
핵심 목표: 데이터 정제(Outlier/NaN 필터링)를 통한 신뢰성 있는 웨이퍼 2D 맵 및 설비 트렌드 시각화와 결함 판정

2. 초급 학습 목표

  1. AI 대화형 데이터 분석 경험 — Gemini 웹 챗에 CSV를 업로드하고, 프롬프트만으로 이상치/결측치를 자동 탐지하는 흐름을 체험합니다.
  2. 프롬프트 엔지니어링 기초 — 원하는 분석 결과를 도출하기 위해 명확하고 구체적인 지시문을 작성하는 능력을 기릅니다.
  3. 반도체 공정 데이터 이해 — Lot, CVD 설비, 웨이퍼 좌표, 두께 값 등 반도체 계측 데이터 구조를 파악합니다.
  4. HTML 보고서 생성 — Gemini가 생성해 준 HTML 소스를 로컬에 저장하여 완성된 보고서를 열어보는 경험을 합니다.

3. 사용 데이터셋

모든 수강생은 아래 엘립소메터 49점 계측 데이터를 사용합니다. CSV 파일을 다운로드하여 Gemini에 업로드하세요.

CSV 다운로드 링크

  • siox_thickness_data_1.csv — CVD_M02 설비 오목형(Edge-High) 이상 데이터
  • siox_thickness_data_2.csv — CVD_M03 설비 볼록형(Center-High) 이상 데이터

데이터 컬럼 설명

컬럼명타입설명
lot_idObject생산 Lot 번호 (LOT-001 ~ LOT-040)
cvd_machineObject증착 CVD 설비 번호 (CVD_M01 ~ CVD_M03)
meas_timeObject박막 측정 완료 시간 (시계열)
position_xFloat웨이퍼 상의 측정 X 좌표 (mm)
position_yFloat웨이퍼 상의 측정 Y 좌표 (mm)
thickness_valueFloat엘립소메터 측정 SiOx 두께 (nm)

4. 단계별 수행 방법

로컬 코딩 빌드 없이 웹 브라우저의 Gemini Chat(웹 클라이언트)에 직접 CSV 파일을 업로드한 후 프롬프트 엔지니어링만으로 분석합니다.

1
Gemini 접속 및 CSV 업로드

웹 브라우저에서 gemini.google.com에 접속합니다. 채팅창의 파일 첨부(클립) 아이콘을 클릭하여 siox_thickness_data_1.csv 파일을 업로드합니다.

2
데이터 구조 파악 요청

업로드된 CSV의 컬럼 구조, 행 수, 데이터 타입을 파악하고, 기초 통계(평균, 표준편차, 최소/최대값)를 요약해 달라고 지시합니다.

3
이상치 및 결측치 탐지

극단값 이상치(-999.0, 999.9, 0.0, 250.3)와 비어 있는 결측치(NaN)를 감지하고 요약하도록 프롬프트합니다. 어떤 Lot과 설비에서 주로 발생하는지도 함께 분석합니다.

4
Lot별 Uniformity 비교 분석

이상치를 제거한 전후의 Lot별 Uniformity (%) 변화를 비교하는 표를 요청합니다. Uniformity 공식: (Max - Min) / (2 * Mean) * 100

5
CVD_M02 설비 이상 원인 분석

특정 시간대에 CVD_M02 설비에서 비정상적으로 균일도가 치솟은 현상에 대한 원인 분석을 지시합니다. 오목형(Edge-High) 패턴이 나타나는 이유를 설명해 달라고 요청합니다.

6
HTML 보고서 생성 요청

모든 분석 결과와 웨이퍼 2D 맵 컨셉, 개선 조치 방안이 정리된 다크모드 스타일의 단일 파일 보고서 report_beginner.html 소스코드를 요청합니다.

7
로컬 저장 및 확인

Gemini가 제공한 HTML 소스코드를 복사하여 텍스트 에디터(메모장 등)에 붙여넣고, report_beginner.html로 저장한 뒤 브라우저에서 열어 최종 확인합니다.

5. 핵심 도구: Gemini Web Chat

Google Gemini는 Google이 제공하는 AI 대화 서비스입니다. 코딩 없이 자연어 프롬프트만으로 데이터를 분석할 수 있습니다.

접속 방법

  1. 웹 브라우저에서 gemini.google.com에 접속합니다.
  2. Google 계정으로 로그인합니다.
  3. 채팅창 하단의 클립(첨부) 아이콘을 클릭하여 CSV 파일을 업로드합니다.
  4. 한국어로 자유롭게 분석 요청을 입력합니다.

활용 팁

  • 한 번에 하나의 요청만 구체적으로 작성하면 더 정확한 결과를 얻습니다.
  • 결과가 부족하면 "더 자세히", "표로 정리해줘" 등 후속 프롬프트를 보냅니다.
  • HTML 코드 요청 시 "단일 파일", "다크모드", "CSS 포함" 등 조건을 명시합니다.

6. 프롬프트 예시

아래 프롬프트들을 순서대로 Gemini에 붙여넣어 사용하세요.

Step 2: 데이터 구조 파악

첨부된 CSV 파일을 분석해줘.
- 전체 행(row) 수와 컬럼(column) 수를 알려줘
- 각 컬럼의 데이터 타입을 정리해줘
- thickness_value 컬럼의 기초 통계(평균, 표준편차, 최소, 최대, 중앙값)를 계산해줘
- lot_id와 cvd_machine의 고유값(unique) 목록을 보여줘

Step 3: 이상치/결측치 탐지

thickness_value 컬럼에서 아래 이상치와 결측치를 모두 찾아줘:
1. 극단적 이상치: -999.0, 999.9, 0.0, 250.3
2. 비어있는 값(NaN/null)

각각에 대해:
- 총 몇 건인지
- 어떤 lot_id, cvd_machine에서 주로 발생하는지
- 발생 시간대(meas_time) 범위
를 표로 정리해줘.

Step 4: Uniformity 비교

Lot별로 두께 균일도(Uniformity)를 계산해줘.
공식: Uniformity(%) = (Max - Min) / (2 * Mean) * 100

아래 두 가지 경우를 비교하는 표를 만들어줘:
1. 이상치 제거 전 (원본 데이터)
2. 이상치(-999.0, 999.9, 0.0, 250.3)와 NaN 제거 후

cvd_machine별로 구분해서 보여주고,
어느 설비에서 Uniformity 차이가 가장 큰지 분석해줘.

Step 5: 설비 이상 원인 분석

CVD_M02 설비에서 이상치가 집중된 시간대를 분석해줘.
- 해당 시간대에 측정된 웨이퍼의 두께 분포 패턴을 설명해줘
- 웨이퍼 중심(Center) 대비 가장자리(Edge) 두께가 어떻게 다른지 분석해줘
- "오목형(Edge-High)" 패턴이란 무엇이고, CVD 설비에서 왜 발생하는지 설명해줘
- 이를 개선하기 위한 조치 방안 3가지를 제안해줘

Step 6: HTML 보고서 생성

지금까지 분석한 내용을 모두 포함하는 완성된 HTML 보고서를 만들어줘.

요구사항:
- 파일명: report_beginner.html
- 단일 HTML 파일 (외부 의존성 없이 CSS 인라인 포함)
- 다크모드 스타일 (배경 #0d1117, 텍스트 흰색 계열)
- 반응형 디자인 (모바일 지원)
- 포함할 섹션:
  1. 프로젝트 제목 및 개요
  2. 데이터 구조 요약 표
  3. 이상치/결측치 탐지 결과 표
  4. 이상치 제거 전후 Uniformity 비교 표
  5. CVD_M02 설비 이상 원인 분석
  6. 웨이퍼 2D 맵 컨셉 설명 (HTML 테이블로 시각적 표현)
  7. 개선 조치 방안
  8. 결론

전체 HTML 소스코드를 출력해줘.

9. 트러블슈팅 (FAQ)

Q. Gemini가 CSV 파일을 인식하지 못해요

A. 파일 크기가 너무 크면 인식이 안될 수 있습니다. CSV를 텍스트 에디터로 열어 상위 100행만 복사하여 텍스트로 붙여넣기를 시도해 보세요. 또한 파일 확장자가 .csv인지 확인하세요.

Q. Gemini가 HTML 코드를 중간에 잘라요

A. 출력 길이 제한 때문입니다. "이어서 계속 출력해줘"라고 입력하거나, 보고서를 섹션별로 나눠서 요청한 뒤 합치세요.

Q. HTML 파일을 저장했는데 브라우저에서 안 열려요

A. 파일을 저장할 때 인코딩을 UTF-8로 지정하고, 확장자가 .html인지 확인하세요. 메모장에서 저장 시 "모든 파일(*.*)"을 선택해야 합니다.

Q. Uniformity 공식이 이해가 안 돼요

Uniformity (%) = (Max - Min) / (2 × Mean) × 100

한 웨이퍼 내에서 가장 두꺼운 곳과 가장 얇은 곳의 차이를 평균 두께의 2배로 나눈 값입니다. 값이 작을수록 균일하게 증착된 것이고, 클수록 불균일합니다. 일반적으로 2% 이하가 양호합니다.

Q. 오목형(Edge-High)이 뭔가요?

A. 웨이퍼 가장자리(Edge)의 박막 두께가 중심(Center)보다 높은 패턴입니다. CVD 챔버의 가스 유량 분배 불균형, 히터 온도 편차, 또는 샤워헤드(Showerhead) 막힘 등이 원인일 수 있습니다.
