Ellipsometer 계측 과정에서 발생한 극단적 노이즈(이상치)와 유실된 계측 데이터(결측치)를 정제하고, 웨이퍼 상의 Film Uniformity를 계산하여 비정상 CVD 설비와 공정 이상 원인을 진단합니다. 9. Troubleshooting / FAQ9. Troubleshooting / FAQ

Q: Gemini가 CSV 파일을 읽지 못해요Q: Gemini cannot read the CSV file

A: 파일 크기가 너무 크면 처음 1000행만 복사/붙여넣기 하거나, "이 CSV의 처음 500행만 분석해줘"라고 요청하세요. 파일 업로드가 안 될 경우 텍스트로 직접 붙여넣기도 가능합니다.A: If the file is too large, copy/paste only the first 1000 rows, or ask "Analyze only the first 500 rows of this CSV." You can also paste data as text if file upload fails.

Q: Uniformity 공식이 헷갈려요Q: I'm confused about the Uniformity formula

A: Uniformity(%) = (Max - Min) / (2 * Mean) * 100입니다. 값이 작을수록 균일하다는 뜻입니다. 이상치를 제거하지 않으면 비현실적으로 높은 값이 나옵니다.A: Uniformity(%) = (Max - Min) / (2 * Mean) * 100. Lower values mean more uniform. Without removing outliers, you'll get unrealistically high values.

Q: HTML 보고서에 차트가 안 나와요Q: Charts are not showing in the HTML report

A: Gemini가 생성한 코드에서 Chart.js CDN 링크가 포함되어 있는지 확인하세요. 인터넷 연결 없이도 작동하려면 인라인 SVG 방식을 요청하세요.A: Check if the Gemini-generated code includes a Chart.js CDN link. For offline use, request inline SVG visualizations instead.

Q: 이상치 기준이 명확하지 않아요Q: Outlier criteria are not clear

A: SiOx 박막 두께의 정상 범위는 약 95~105nm입니다. -999, 999.9, 0.0은 센서 오류값이고, 250.3nm처럼 정상 범위를 크게 벗어나는 값도 이상치로 처리합니다.A: Normal SiOx film thickness range is approx. 95~105nm. Values like -999, 999.9, 0.0 are sensor errors, and values far outside range like 250.3nm are also treated as outliers.
