프로젝트 1 — 반도체 CVD 박막 두께 Wafer Map 데이터 분석

1. 프로젝트 개요

실제 반도체 양산 공정에서 널리 쓰이는 PECVD 실리콘 산화막(SiOx) 박막 증착 두께 데이터를 분석하는 실무 프로젝트입니다.

엘립소메터(Ellipsometer) 계측 과정에서 발생한 극단적 노이즈(이상치)와 유실된 계측 데이터(결측치)를 정제하고, 웨이퍼 상의 박막 균일도(Uniformity)를 계산하여 비정상 CVD 설비와 공정 이상 원인을 진단합니다.

수행 기간: CH.1 이후 주말 2일 (토~일)
핵심 목표: 어떤 실험 파일을 넣어도 동작하는 재현 가능한(Reproducible) Streamlit 대시보드 파이프라인 구축

2. 고급 학습 목표

  1. Streamlit 대시보드 설계 — Python 기반 데이터 대시보드 프레임워크를 사용하여 전문적인 분석 도구를 구축합니다.
  2. 동적 전처리 파이프라인 — 파일 업로더로 어떤 CSV를 넣어도 자동으로 Lot/설비를 식별하고 정제하는 범용 파이프라인을 설계합니다.
  3. Plotly Express 고급 시각화 — 300mm 원형 웨이퍼 산점도 맵을 반응형으로 구현하고, Uniformity 추이를 인터랙티브 차트로 표현합니다.
  4. 재현성 검증 — data_1(오목형)과 data_2(볼록형) 두 파일 모두 정상적으로 분석되는지 검증합니다.
  5. Antigravity + Streamlit 워크플로우 — AI 에이전트를 활용하여 Python 코드를 점진적으로 완성하고 디버깅하는 실무 능력을 기릅니다.

3. 사용 데이터셋

고급에서는 두 가지 데이터셋 모두를 사용하여 재현성을 검증해야 합니다.

CSV 다운로드 링크

  • siox_thickness_data_1.csv — CVD_M02 설비 오목형(Edge-High) 이상 데이터
  • siox_thickness_data_2.csv — CVD_M03 설비 볼록형(Center-High) 이상 데이터
재현성 필수 조건: 완성된 대시보드에 data_1을 업로드하면 CVD_M02 오목형 이상이, data_2를 업로드하면 CVD_M03 볼록형 이상이 자동으로 감지/시각화되어야 합니다.

4. 단계별 수행 방법

Antigravity를 활용하여 Python 기반 Streamlit 애플리케이션을 완성합니다.

1
개발 환경 설정

Python 가상 환경을 생성하고 필요한 패키지를 설치합니다. requirements.txt를 작성합니다.

# 가상 환경 생성 및 활성화
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 패키지 설치
pip install streamlit pandas numpy plotly scipy
2
Streamlit 대시보드 골격 설계

Antigravity에게 app.py의 기본 구조를 제안받습니다. 사이드바, 파일 업로더, 메인 영역의 레이아웃을 설계합니다.

3
범용 파일 업로더 및 자동 식별

st.file_uploader로 CSV를 업로드하면 자동으로 Lot 데이터, 설비 목록, 계측점 수를 식별하여 대시보드가 리프레시되는 코드를 작성합니다.

4
이상치 필터링 및 결측치 보간 옵션

사이드바에 체크박스/슬라이더 위젯을 배치하여 사용자가 직접 전처리 옵션을 조절할 수 있게 합니다.

  • st.checkbox: 이상치 필터링 ON/OFF
  • st.multiselect: 이상치 값 선택 (-999.0, 999.9, 0.0, 250.3)
  • st.radio: 보간 방법 선택 (그룹 평균 / 중앙값 / 보간 없음)
  • st.slider: 정상 두께 범위 필터 (예: 800~1200nm)
5
Plotly 2D 원형 웨이퍼 산점도 맵

Plotly Express의 px.scatter를 활용하여 동심원 측정 좌표(X, Y) 기반의 2D 원형 웨이퍼 산점도 맵을 반응형으로 구현합니다.

  • 300mm 웨이퍼 외곽 원형 표시 (shape annotation)
  • 두께 값에 따른 연속 색상 매핑 (Viridis 또는 RdYlBu)
  • Lot 선택에 따른 실시간 맵 업데이트
  • 호버 툴팁: 좌표, 두께, Lot, 설비 정보 표시
6
Uniformity 계산 및 설비별 트렌드 차트

Uniformity 공식을 적용한 계산 로직과 함께, 설비별/Lot별 추이를 Plotly line chart로 구현합니다. 이상 구간을 자동 감지하여 하이라이트합니다.

7
재현성 검증

2개의 상이한 데이터를 대입하여 검증합니다:

  • data_1 업로드 → CVD_M02 오목형(Edge-High) 패턴이 감지되는지 확인
  • data_2 업로드 → CVD_M03 볼록형(Center-High) 패턴이 감지되는지 확인
  • 두 경우 모두 대시보드가 정상 동작하고, 이상 설비가 자동 식별되는지 최종 검증
8
requirements.txt 작성 및 실행 가이드

프로젝트를 다른 환경에서도 재현할 수 있도록 requirements.txt와 실행 명령을 문서화합니다.

6. 프롬프트 예시 (Antigravity용)

아래 프롬프트들을 Antigravity 에이전트에게 순서대로 전달하세요.

Step 2: Streamlit 골격 설계

Streamlit 대시보드 app.py를 만들어줘.

프로젝트: 반도체 CVD 박막 두께 Wafer Map 데이터 분석
데이터: 엘립소메터 49점 계측 CSV (lot_id, cvd_machine, meas_time, position_x, position_y, thickness_value)

기본 구조:
1. 페이지 설정: 와이드 레이아웃, 다크 테마, 제목 "CVD 박막 두께 분석 대시보드"
2. 사이드바: 파일 업로더(CSV), 전처리 옵션 (이상치 필터링 체크박스, 보간 방법 라디오)
3. 메인 영역 상단: KPI 메트릭 카드 4개 (총 Lot수, 설비수, 평균 두께, 평균 Uniformity)
4. 메인 영역 중단: 2열 레이아웃 (좌: Wafer Map, 우: 설비별 박스플롯)
5. 메인 영역 하단: Lot별 트렌드 라인 차트 (전체 너비)

일단 빈 플레이스홀더로 구조만 잡아줘.

Step 3: 범용 파일 업로더

st.file_uploader로 CSV를 업로드하면 자동으로 데이터를 식별하는 코드를 만들어줘.

요구사항:
- pd.read_csv로 파싱
- lot_id 고유값 목록과 개수 자동 추출
- cvd_machine 고유값 목록 자동 추출
- thickness_value 기초 통계 자동 계산
- 데이터가 없으면 안내 메시지 표시
- 어떤 CSV를 넣어도 동작하도록 컬럼명 검증 로직 포함
  (필수 컬럼 누락 시 에러 메시지)

Step 4: 전처리 옵션 위젯

사이드바에 전처리 옵션 위젯을 추가해줘.

위젯 목록:
1. st.checkbox("이상치 필터링 적용") - 기본값 True
2. st.multiselect("제거할 이상치 값", [-999.0, 999.9, 0.0, 250.3], default=[-999.0, 999.9, 0.0, 250.3])
3. st.radio("결측치 보간 방법", ["그룹 평균", "그룹 중앙값", "보간 없음(NaN 유지)"])
4. st.slider("정상 두께 범위 (nm)", min_value=0, max_value=2000, value=(800, 1200))

위젯 값에 따라 데이터프레임을 동적으로 필터링/보간하는 함수를 만들어줘.
그룹은 같은 lot_id + cvd_machine 조합이야.

Step 5: Plotly 웨이퍼 맵

Plotly Express로 300mm 웨이퍼 2D 산점도 맵을 그려줘.

요구사항:
- px.scatter(x=position_x, y=position_y, color=thickness_value)
- 색상 스케일: "RdYlBu_r" (빨강=두꺼움, 파랑=얇음)
- 마커 크기: 12~15
- 300mm 웨이퍼 외곽 원형 표시 (fig.add_shape으로 반경 150mm 원)
- X/Y축 범위: -160 ~ 160
- 축 비율 1:1 (scaleanchor)
- 호버 툴팁: position_x, position_y, thickness_value, lot_id, cvd_machine
- 다크 테마 (template="plotly_dark")
- Lot 선택 st.selectbox 연동: 선택한 Lot만 표시
- st.plotly_chart(fig, use_container_width=True)

Step 6: Uniformity 트렌드 차트

Lot별 + 설비별 Uniformity 추이를 Plotly line chart로 그려줘.

Uniformity 공식: (Max - Min) / (2 * Mean) * 100
각 Lot + cvd_machine 조합별로 계산

요구사항:
- X축: lot_id
- Y축: Uniformity (%)
- 색상: cvd_machine별 구분 (M01=파랑, M02=빨강, M03=초록)
- 마커 + 라인 (mode="lines+markers")
- 이상 Uniformity 기준선 표시 (2% 수평선, 빨간 점선)
- 다크 테마
- 하단에 평균 두께 트렌드도 서브플롯으로 추가

Step 7: 재현성 검증

재현성 검증을 위한 자동 진단 섹션을 추가해줘.

데이터를 업로드하면 자동으로:
1. 가장 Uniformity가 높은(불균일한) 설비를 식별
2. 해당 설비의 웨이퍼 맵 패턴을 분석
   - Center vs Edge 두께 평균 비교 (반경 50mm 이내 vs 100mm 이상)
   - 패턴 판정: "오목형(Edge-High)" 또는 "볼록형(Center-High)"
3. 진단 결과를 st.success 또는 st.warning 박스로 표시
4. 예: "CVD_M02 설비에서 오목형(Edge-High) 이상 패턴이 감지되었습니다.
   Center 평균: 998.5nm, Edge 평균: 1005.2nm, 차이: +6.7nm"

data_1 -> CVD_M02 오목형, data_2 -> CVD_M03 볼록형이 나와야 해.

9. 트러블슈팅 (FAQ)

Q. Streamlit 설치가 안 돼요

A. Python 3.8 이상이 필요합니다. python --version으로 확인하세요. pip가 오래된 경우 pip install --upgrade pip 후 재시도합니다.

# Python 버전 확인
python --version

# pip 업그레이드 후 설치
pip install --upgrade pip
pip install streamlit pandas numpy plotly

Q. st.file_uploader가 파일을 읽지 못해요

A. st.file_uploader는 UploadedFile 객체를 반환합니다. pd.read_csv(uploaded_file)로 직접 읽을 수 있습니다.

uploaded_file = st.file_uploader("CSV 업로드", type="csv")
if uploaded_file is not None:
    df = pd.read_csv(uploaded_file)
    st.dataframe(df.head())

Q. Plotly 웨이퍼 맵에서 원이 타원형으로 나와요

A. X/Y축 비율을 1:1로 맞춰야 합니다. fig.update_yaxes(scaleanchor="x", scaleratio=1)을 추가하세요.

fig = px.scatter(df, x="position_x", y="position_y",
                 color="thickness_value", color_continuous_scale="RdYlBu_r")
fig.update_yaxes(scaleanchor="x", scaleratio=1)
fig.update_layout(
    xaxis_range=[-160, 160],
    yaxis_range=[-160, 160],
    template="plotly_dark"
)
# 웨이퍼 외곽 원 추가
fig.add_shape(type="circle", x0=-150, y0=-150, x1=150, y1=150,
              line=dict(color="gray", width=2))

Q. 재현성 검증에서 오목/볼록 판정이 안 맞아요

A. Center(반경 50mm 이내)와 Edge(반경 100mm 이상) 영역을 정확히 구분해야 합니다. 이상치 제거 후의 정제 데이터로 판정하세요.

# 반경 계산
df['radius'] = np.sqrt(df['position_x']**2 + df['position_y']**2)

# Center vs Edge 영역 평균
center_mean = df[df['radius'] <= 50]['thickness_value'].mean()
edge_mean = df[df['radius'] >= 100]['thickness_value'].mean()

# 판정
if edge_mean > center_mean:
    pattern = "오목형(Edge-High)"  # 가장자리가 더 두꺼움
else:
    pattern = "볼록형(Center-High)"  # 중심이 더 두꺼움

Q. Streamlit 앱이 자꾸 새로고침돼요

A. Streamlit은 코드 변경 시 자동 새로고침됩니다. @st.cache_data 데코레이터를 사용하면 데이터 로딩을 캐싱하여 불필요한 재로딩을 방지할 수 있습니다.
