9. Troubleshooting / FAQ9. Troubleshooting / FAQ

Q: Streamlit이 설치가 안 돼요Q: Streamlit won't install

A: Python 3.8 이상이 필요합니다. pip install streamlit plotly pandas numpy를 실행하세요. 가상환경(venv) 사용을 권장합니다: python -m venv venv && source venv/bin/activateA: Python 3.8+ is required. Run pip install streamlit plotly pandas numpy. Using a virtual environment is recommended: python -m venv venv && source venv/bin/activate

Q: Plotly 차트가 Streamlit에서 안 보여요Q: Plotly chart not showing in Streamlit

A: st.plotly_chart(fig, use_container_width=True)를 사용하세요. fig.show()는 Streamlit에서 작동하지 않습니다. plotly 버전이 5.0 이상인지 확인하세요.A: Use st.plotly_chart(fig, use_container_width=True). fig.show() doesn't work in Streamlit. Check that plotly version is 5.0+.

Q: 파일 업로드 후 대시보드가 업데이트 안 돼요Q: Dashboard doesn't update after file upload

A: Streamlit은 위젯 상태 변경 시 전체 스크립트를 재실행합니다. @st.cache_data 데코레이터로 데이터 로딩 함수를 캐싱하고, 세션 상태를 적절히 관리하세요.A: Streamlit reruns the entire script on widget state change. Cache data loading functions with @st.cache_data decorator and manage session state properly.

Q: 재현성 검증에서 설비 자동 식별이 안 돼요Q: Equipment auto-identification fails in reproducibility test

A: 설비별 Uniformity 평균을 비교하여 가장 높은 설비를 이상 설비로 판정하세요. 또는 설비별 Uniformity의 표준편차가 가장 큰 설비를 선택합니다. 임계값(threshold)은 전체 평균 + 2*표준편차를 사용하세요.A: Compare per-equipment Uniformity means and flag the highest as anomalous. Alternatively, select the equipment with the largest Uniformity standard deviation. Use threshold of overall mean + 2*std.

Q: 오목형/볼록형 패턴 구분이 어려워요Q: Hard to distinguish concave/convex patterns

A: 웨이퍼 중심부(r < 50mm)와 가장자리(r > 100mm) 영역의 평균 두께를 비교하세요. 중심 < 가장자리이면 오목형(Edge-High), 중심 > 가장자리이면 볼록형(Center-High)입니다.A: Compare average thickness of center region (r < 50mm) vs edge region (r > 100mm). Center < Edge = concave (Edge-High), Center > Edge = convex (Center-High).
